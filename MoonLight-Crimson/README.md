# MoonLight Crimson

Aggressive AMOLED crimson theme for Kettu/Rain.

## Features

- True AMOLED black
- Crimson glow
- Blur panels
- Floating cards
- Minimal inputs
- Smooth animations
- Themes+ support
- Custom overlays
- JetBrains Mono chat font

## Requirements

- Kettu or Rain
- Themes+ enabled

## Install

Import:
MoonLight-Crimson.json

Optional CSS:
crimson.css

## Recommended

- You Bar experiment
- AMOLED mode
- Transparent status bar
